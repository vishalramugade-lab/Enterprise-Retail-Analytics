## Project Objective

The objective of this project is to build a centralized Azure-based analytics platform that integrates data from multiple enterprise systems into a unified reporting solution.

The platform will support executive decision-making through interactive Power BI dashboards while improving data quality, reporting performance, and governance.

---

## Existing Challenges

• Data spread across multiple systems

• Manual Excel reporting

• Different KPI definitions

• No centralized data warehouse

• Poor report performance

• No historical data tracking

• Difficult executive reporting

• Duplicate customer records

• Inconsistent product information

---

## Proposed Solution

Implement a Microsoft Azure Modern Data Platform using:

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- Azure SQL Database
- Power BI
- Azure DevOps
- Power BI Embedded

---
